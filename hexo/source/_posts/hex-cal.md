# 以查看/proc/2206/maps下的so大小为例进行说明，计算方法如下
```
cat /proc/2206/maps | awk '{print $1}' | awk -F '-' '{printf "echo \"obase=16\;\$\(\(0x%s-0x%s\)\)\"|bc\n", $2, $1}'
```
